### Вопросы к заданию

**В чём разница между setTimeout() и setinterval()? Когда использовать каждый?** SetTimeout принимает колбэк и время через которое он будет передан в макрозадачи очереди. Колбэк выполняется лишь раз. Setinterval тоже принимает колбэк и время, но этот колбэк будет выполняться множество раз через указанное время.

**Что произойдёт, если не вызвать clear Interval()? Какие проблемы это может вызвать?**
Если не вызвать clearinterval(), setinterval будет выполняться бесконечное кол-во раз. Что приведёт к мемори лик. 

**Что такое ВОМ? Назовите основные объекты ВОМ и их назначение.**
Объектная модель браузера, предоставляющая js интерфейсы для взаимодействия с самим браузером, а не содержимым веб страницы. 
Основные объекты: 
* navigator (информация о браузере) его версиях, операционной системе, плагинах
* screen (разрешение экрана) данные об экране монитора(ширина, высота)
* location (url и навигация) позволяет управлять адресной строкой, получать текущий url или перенаправлять пользователя на другую страницу 
* history (история посещений) дает возможность перемещаться по истории переходов.
* localStorage (хранение данных)
* document (Dom - содержимое страницы)
* Таймеры (setTimeout, setinterval)

**Как узнать, с мобильного ли устройства зашёл пользователь? Приведите пример кода.**
Используя navigator.userAgent 

**Почему alert(), confirm() и prompt() считаются устаревшими? Какие у них недостатки?**
Основные недостатки:   
* Блокируют полностью выполнение js
* Невозможность стилизации
* Блокировка взаимодействий пользователя со страницей
* Лучше использовать кастомные модальные окна. 

**Что делает метод padStart()? Зачем он нужен при работе с таймерами?**
Дополняет строку другой строкой ,указанными символами в начале до достижения указанной длины строки. В таймере добавляет 0 перед однозначными цифрами для обеспечения двузначного формата(00:00).


**Как сохранить состояние таймера в localStorage , чтобы при перезагрузке страницы таймер продолжал работать?**
Сохранить объект в котором хранится время таймера и булеан состояние таймера (isRunning). При перезагрузке страницы загружать объект из localstorage и проверять свойства isRunning. Если true, то возобновить таймер.


**Что такое location.href? Как перенаправить пользователя на другую страницу через JavaScript?** 
Свойство, содержащее полный URl-адрес текущей страницы. При присваивании нового значения этому свойству браузер будет осуществлять переход по этому адресу, что вызовет загрузку новой страницы.
